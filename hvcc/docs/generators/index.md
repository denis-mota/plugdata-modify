# Generators

HVCC supports a number of dedicated generators that can help to wrap the heavy c/cpp output into a pluggable format:

* [Daisy](daisy.md)
* [DPF](dpf.md)
* [FMOD](fmod.md)
* [Javascript](javascript.md)
* [OWL](owl.md)
* [Pdext](pdext.md)
* [Wwise](wwise.md)
* [Unity](unity.md)

You can also create your own generator without modifying HVCC, see [External Generators](custom.md).
